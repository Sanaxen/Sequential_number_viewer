# SEQ VIEWER — 連番画像ビューワ

連番画像のアニメーション再生・コマ送り・動画変換ができるローカルWEBアプリです。

---

手作業一切なし
## プロンプト（Claude Sonnet4.6)  

``
連番画像ビューワを作ってほしい。  
- python  
- WEBアプリ（ローカルホスト）  
- ライブラリはアプリを収納するディレクトリ内にディレクトリを作ってそこに集約  
- バッチでライブラリなどインストールから実行（インストール済なら即実行）  
- 連番画像をアニメーション（パラパラ漫画）できる  
- コマ送りもできる  
- スライダーで自由にアニメ  
- 頭出し、最後への移動がボタン一発  
- 連番は整数で不定間隔で飛び飛びの場合もある  
- 連番は0001とか01とか1のような色んな形式があるがユーザーは意識しなくていい。  
- 連番は画像ファイル名に含まれている。必ずファイル名の最後が連番でaaa01やaaa_01とかいろいろとあり得る。aaaの部分は共通  
- 画像形式はbmp,jpg,png  
・アプリ内に表示するがサイズはアプリのサイズに連動して固定サイズ表示しない  
- 動画変換機能がある  
- 画像ファイルは逐次作成されたり更新される可能性があることを考慮する  
``
---

## 起動方法

### Windows
```
start.bat をダブルクリック
```

### macOS / Linux
```bash
chmod +x start.sh
./start.sh
```

ブラウザで `http://localhost:5000` が開きます。

---

## 機能

| 機能 | 説明 |
|------|------|
| 連番画像読み込み | ディレクトリパスを入力して LOAD |
| アニメーション再生 | ▶ ボタン または スペースキー |
| コマ送り | ◀ ▶ ボタン または 矢印キー / `,` `.` |
| 頭出し / 最後 | ⏮ ⏭ ボタン または Home/End キー |
| スライダー | 自由にフレーム移動 |
| ループ | ∞ ボタンで ON/OFF |
| FPS調整 | PLAY行の FPS 欄で変更 |
| ライブ更新 | 2秒ごとに新規/更新ファイルを自動検出 |
| 動画変換 | VIDEO ボタンで MP4/GIF/WEBM に変換 |

---

## 対応フォーマット

- 画像: `.bmp`, `.jpg`, `.jpeg`, `.png`
- 動画出力: `.mp4`, `.gif`, `.webm`（要 ffmpeg）

---

## 連番ファイル名の仕様

以下のような様々な形式に対応しています：

- `frame001.png`, `frame01.png`, `frame1.png`
- `aaa_001.jpg`, `aaa001.bmp`
- `render0001.png`

**条件**: ファイル名の末尾（拡張子の直前）が連番数字であること。  
飛び番（001, 003, 007...）も正しく並べて表示します。

---

## 動画変換について

ffmpeg が必要です。

- **Windows**: https://ffmpeg.org/download.html
- **macOS**: `brew install ffmpeg`
- **Linux**: `sudo apt install ffmpeg`

出力ファイルは画像と同じディレクトリに保存されます。
